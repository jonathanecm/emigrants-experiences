# emigrants-experiences
An computational context analysis on the experiences of emigrants from Venezuela.


This project uses:
* Python script for normalization, tokenizations, and lemmatization using tool from nltk.
* Python script for a parts of speech analysis.
* A conditional probability distribution with a total of 2424 conditions.
* Bigrams, trigrams, and n-grams analysis.
* Tests bias on n-grmas using student_t, chi_sq, likelihood_ratio, and pmi statistics.
* Techniques of information extraction, and computation and linguistic models to parse precise claims from the interviews.
* Named Entity Recognition (NER) to identify object and entities.
* Topic modeling models such LDA or PCA.
* Hierarchical clustering Wald's method and Cosine Similarity.
* Semantic networks and word embeddings techniques Centrality and Betweeness.
